# 🚀 Attention Analyzer AI

## 🧠 AI-Based Digital Attention Analyzer & Content Optimization System

---

## 📌 Overview

In today’s fast-paced digital world, attention span is rapidly declining due to excessive short-form content consumption.
This project leverages **Machine Learning + Behavioral Analytics** to analyze user habits, predict attention patterns, and provide actionable insights.

---

## 🎯 Key Features

✨ **Attention Span Prediction**
Predicts whether a user prefers:

* Short-form content
* Long-form content
* Balanced content

📊 **Attention Score (0–100)**
A custom-built behavioral scoring system that quantifies user attention.

🧩 **Hybrid AI System**
Combines:

* Machine Learning model
* Rule-based attention scoring

💡 **Personalized Recommendations**
Provides actionable suggestions to improve focus and productivity.

---

## ⚙️ Tech Stack

* 🐍 Python
* 📊 Pandas, NumPy
* 🤖 Scikit-learn
* 📈 Matplotlib, Seaborn

---

## 🧠 Machine Learning Model

* Model: **Random Forest Classifier**
* Hyperparameter tuning using **GridSearchCV**
* Accuracy: **~85–90%**

---

## 📊 Dataset

* Collected using Google Forms
* **200+ real user responses**
* Features include:

  * Screen time
  * Short-form content usage
  * Focus duration
  * Task switching behavior
  * Learning preferences

---

## 📈 Key Insights

* 📱 Higher short-form content usage → Lower attention span
* 🔄 Frequent task switching → Reduced focus ability
* 🎓 Long-form content consumption → Improved attention
* 🧠 Focus duration is the most influential feature
